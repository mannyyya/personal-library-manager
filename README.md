# 📚 Personal Library Manager 

A Python console-based application for managing a personal library.

## Features

* Add Books
* View Books
* Search Books
* Delete Books
* Update Book Information
* Save Library to File
* Load Library from File
* Count Total Books
* View Books by Genre

## Concepts Used

* Python Functions
* Lists
* Dictionaries
* File Handling
* CRUD Operations
* Search Algorithms
* Frequency Tables

## How to Run

```bash
python library_manager.py
```
